# React Tutorials

This repository contains a collection of React tutorials and examples, organized into different sections to help you learn React step by step. Each tutorial is a separate Create React App project, demonstrating various concepts and features of React.

## Prerequisites

Before running any of the tutorials, make sure you have the following installed:

- [Node.js](https://nodejs.org/) (version 8 or higher)
- [npm](https://www.npmjs.com/) (comes with Node.js)

## Getting Started

1. Clone or download this repository.
2. Navigate to the specific tutorial folder you want to run.
3. Install dependencies: `npm install`
4. Start the development server: `npm start`
5. Open [http://localhost:3000](http://localhost:3000) in your browser to view the app.

## Tutorials

### React Fundamentals

This section covers the core concepts of React, including components, state, props, lifecycle methods, and advanced patterns.

#### hello-world

- **Description**: A basic "Hello World" React application created with Create React App.
- **Concepts**: Setting up a React project, basic component structure.

#### components-demo

- **Description**: Demonstrates different types of React components and their usage.
- **Concepts**: Functional components, class components, props, state, event handling, lists and keys, conditional rendering.

#### forms-demo

- **Description**: Shows how to handle forms in React.
- **Concepts**: Controlled components, form submission, handling multiple inputs.

#### lifecycle-demo

- **Description**: Illustrates React component lifecycle methods.
- **Concepts**: Mounting, updating, and unmounting lifecycle methods.

#### advanced-guide-demo

- **Description**: Covers advanced React concepts and patterns.
- **Concepts**: Fragments, refs, portals, error boundaries, higher-order components (HOCs), render props, context API.

#### styling-demo

- **Description**: Demonstrates various ways to style React components.
- **Concepts**: CSS stylesheets, inline styles, CSS modules.

### React Hooks

This section introduces React Hooks, which allow you to use state and other React features in functional components.

#### state-hook

- **Description**: Introduction to the useState hook.
- **Concepts**: Managing state in functional components, updating state, multiple state variables.

#### effect-hook

- **Description**: Covers the useEffect hook for side effects.
- **Concepts**: Running effects on mount/update/unmount, cleanup functions, conditional effects.

#### context-hook

- **Description**: Demonstrates the useContext hook.
- **Concepts**: Consuming context in functional components.

#### reducer-hook

- **Description**: Shows how to use the useReducer hook.
- **Concepts**: Complex state logic, reducer pattern in React.

#### callback-hook

- **Description**: Covers the useCallback hook for performance optimization.
- **Concepts**: Memoizing functions to prevent unnecessary re-renders.

#### memo-hook

- **Description**: Demonstrates the useMemo hook.
- **Concepts**: Memoizing expensive computations.

#### ref-hook

- **Description**: Introduces the useRef hook.
- **Concepts**: Accessing DOM elements, persisting values across renders.

#### custom-hooks

- **Description**: Shows how to create and use custom hooks.
- **Concepts**: Extracting reusable logic into custom hooks.

### React and HTTP

This section covers making HTTP requests in React applications.

#### react-http

- **Description**: Demonstrates fetching data from APIs using React.
- **Concepts**: Making GET and POST requests, handling loading states, error handling.
